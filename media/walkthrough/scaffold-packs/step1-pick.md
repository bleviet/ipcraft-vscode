## Choose your starting point

IPCraft ships five packs ready to export:

| Pack | What it generates |
|------|-------------------|
| **builtin-minimal** | One top-level entity/module stub with an empty body |
| **builtin-ipcraft** | Package · Top · Core · Bus wrapper · Register file |
| **example-register-only** | Only the register file decoder |
| **example-no-regfile** | Package · Top · Core · Bus wrapper (no register file) |
| **example-with-docs** | Full layered set + a Markdown register reference |

Each example pack's `scaffold.yml` is heavily commented to explain every field.
Start with the one closest to what you need, then customise from there.
