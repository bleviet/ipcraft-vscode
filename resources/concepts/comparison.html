IPCraft/Issue #41/Icon redesign

Three marks, three minds.

The current icon dates from when the Memory Map was the whole product — it reads as a register table. The IP Core canvas is now an equal pillar, so each concept below reframes the mark as an IP block with interfaces. Three agents drafted independently, blind to one another, one per model.

Replacing Current icon — three register rows with a highlighted bitfield. Half the story: no interfaces, no block.
3 concepts 3 models · blind brand gradient kept

Connected Core

A gradient chip with pins on three edges and one interface routed out to a terminal ring.

At sidebar size — 40px · 32px

The one-out asymmetry gives it a silhouette no generic chip icon has — block plus wired node — and states the canvas directly. Registers survive only as die-cut texture.

  • IdiomTransparent glyph
  • FillSolid gradient, negative-space cuts
  • Ports3 edges + 1 routed to ring
  • DownscaleZero strokes — cannot thin out
Trade-offMost ownable silhouette; but the outbound ring can read as an antenna/tail at 32px.

Core & Ports

A white IP module on the brand tile, interface ports each side, one bitfield lit amber.

At sidebar size — 40px · 32px

The boldest, most finished of the three — a shipped-product tile. White-on-gradient contrast holds at any size; the amber field is a warm focal spark and a nod to the old highlight.

  • IdiomFull brand tile
  • FillWhite module, knocked-out cells
  • Ports3 per side (L / R)
  • AccentAmber active bitfield
Trade-offReads instantly at every size; but it is a full tile (unlike today's glyph) and its face still leans register-forward.

Portcore

An outlined chip block with pads on all four sides and a faint register grid inside.

At sidebar size — 40px · 32px

The lightest, most minimal read — pins and block form one continuous line system, with the register grid fading to soft texture. Elegant and quiet.

  • IdiomTransparent glyph
  • FillOutline / stroke, one solid cell
  • Ports4 sides, round pads
  • DownscaleStrokes thin out; grid softest
Trade-offMost refined line-work; but the outline + faint grid is the most fragile at 32px and closest in spirit to the old icon.

Picking one

The real fork is full tile (B) versus transparent glyph (A, C) — and bold (B) versus minimal (C), with A the most distinctive silhouette between them.

All three live on branch icon-concepts under resources/concepts/ (SVG source + rationale + 128px PNG), committed but not pushed.
