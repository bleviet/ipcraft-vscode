# icon-gpt54

Concept direction: a single IP-module glyph with explicit edge ports and a structured internal layout.

- Primary read: an FPGA IP block on the canvas.
- Secondary read: ordered register / field lanes inside the block.
- Distinguishing choice: offset utility ports plus a wider bus-style interface, with an internal block-diagram layout instead of a literal register table.

Files:

- `icon-gpt54.svg` — source of truth
- `icon-gpt54-128.png` — 128x128 raster export
